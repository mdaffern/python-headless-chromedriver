# python-headless-chromedriver

This is a Docker container with python, it uses chrome as driver
See [Dockerfile](https://github.com/).

### Dependencies
You can change the requirements, it currently installs:
xvfbwrapper 
selenium
requests==2.13.0
beautifulsoup4 
numpy==1.11.2
boto==2.45.0
boto3==1.4.3
smart_open
